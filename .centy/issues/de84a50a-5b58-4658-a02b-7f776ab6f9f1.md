# Make button 15 a context-aware search button

## Overview

Button 15 should function as a search button, but only be enabled in contexts where search is relevant (e.g., Centy projects list, issues list). It should be disabled/hidden in screens where search doesn't apply.

## Requirements

- Button 15 triggers a search action when activated
- Button 15 is **not always enabled** — it should be context-aware
- Enable button 15 in screens where search is applicable:
  - Centy projects list
  - Centy issues list
- Disable (or hide) button 15 in screens where search is not applicable

## Acceptance Criteria

- [ ] Button 15 is enabled on the Centy projects screen and triggers search
- [ ] Button 15 is enabled on the Centy issues screen and triggers search
- [ ] Button 15 is disabled on screens that do not support search
- [ ] No action (or a no-op) occurs when button 15 is pressed in a non-search context
