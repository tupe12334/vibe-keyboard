# Make button 14 a context-aware sort button

## Overview

Button 14 should function as a sort button, but only be enabled in contexts where sorting is relevant (e.g., Centy projects list, issues list). It should be disabled/hidden in screens where sorting doesn't apply.

## Requirements

- Button 14 triggers a sort action when activated
- Button 14 is **not always enabled** — it should be context-aware
- Enable button 14 in screens where sort is applicable:
  - Centy projects list
  - Centy issues list
- Disable (or hide) button 14 in screens where sort is not applicable

## Acceptance Criteria

- [ ] Button 14 is enabled on the Centy projects screen and triggers sort
- [ ] Button 14 is enabled on the Centy issues screen and triggers sort
- [ ] Button 14 is disabled on screens that do not support sort
- [ ] No action (or a no-op) occurs when button 14 is pressed in a non-sort context
