# Add input number page with numpad-style keyboard layout

## Overview

Create an input number page that maps keyboard keys to a numpad-style number entry interface.

## Key Mappings

| Physical Keys | Input Digit |
|---------------|-------------|
| 3, 4, 5       | 1, 2, 3     |
| 8, 9, 10      | 4, 5, 6     |
| 13, 14, 15    | 7, 8, 9     |
| 1             | Clear all   |
| 2             | Backspace (delete one char) |

## Acceptance Criteria

- [ ] Keys 3-5 map to digits 1-3
- [ ] Keys 8-10 map to digits 4-6
- [ ] Keys 13-15 map to digits 7-9
- [ ] Key 1 clears the entire input
- [ ] Key 2 deletes the last character (backspace)
- [ ] Display shows current number being entered
